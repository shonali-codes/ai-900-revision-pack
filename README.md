# AI-900 Preparation Support Pack
**Microsoft Azure AI Fundamentals (AI-900)**

This repository contains a **structured, exam-focused AI-900 preparation pack** designed to help learners quickly understand and revise the key concepts tested in the AI-900 exam.

The materials focus on **concept recognition and scenario-based decision-making**, which is exactly what the AI-900 exam evaluates.

---

## 🎯 Objective

To simplify AI-900 preparation by:
- Providing a **big-picture understanding** of the syllabus
- Mapping **exam keywords to the correct Azure AI services**
- Offering **concise revision notes and cheat tables**
- Avoiding unnecessary depth (no coding, math, or implementation details)

---

## 📦 What’s Included

### 1️⃣ Big Picture Map
A visual overview of the entire AI-900 syllabus showing:
- AI fundamentals
- Azure AI Services vs Azure Machine Learning
- Core workloads: Vision, NLP, Speech, Document Intelligence, Generative AI

This helps build **conceptual clarity before detailed revision**.

---

### 2️⃣ Exam-Focused Cheat Tables
Scenario-driven tables covering:

- Keyword → Azure Service mapping  
- Responsible AI principles  
- ML problem types (classification, regression, clustering)  
- Azure AI Services vs Azure Machine Learning decisions  
- ML lifecycle (training vs inferencing)  
- Model evaluation metrics (recognition level only)

These tables are designed for **quick daily revision and exam recall**.

---

### 3️⃣ AI-900 Revision Notes (Consolidated)
A single, structured revision document that:
- Covers the **complete AI-900 syllabus**
- Uses **simple language and bullet points**
- Is ideal for **final-week and last-day revision**

---

## 🧠 Why This Approach Works for AI-900

AI-900 is **not an implementation exam**.  
It primarily tests:
- Recognition of AI workloads
- Correct Azure service selection
- Understanding responsible AI scenarios
- Training vs inference decisions

This repo is intentionally focused on:
✅ Concept clarity  
✅ Scenario recognition  
✅ Exam-aligned depth  

---

## ✅ How to Use This Repo for Preparation

Suggested flow:
1. Start with the **Big Picture Map**
2. Use the **cheat tables** for short, frequent revision
3. Read the **revision notes** during the final days before the exam

If you can explain:
- Which Azure service to use and why
- Which AI principle applies in a scenario
- Whether training is required or not

You are likely **exam-ready for AI-900**.
